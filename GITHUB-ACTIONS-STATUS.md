## 🐛 如果构建失败

### 常见失败原因

1. **测试标题重复**
   ```
   Error: duplicate test title "..."
   ```
   **解决**: 已修复 (`a5bff71`)

2. **E2E 测试超时**
   ```
   Test timeout of 30000ms exceeded
   ```
   **解决**: 增加超时或优化测试

3. **浏览器启动失败**
   ```
   Error: browserType.launch failed
   ```
   **解决**: CI 环境问题，重试或联系 GitHub

4. **依赖安装失败**
   ```
   npm install failed
   ```
   **解决**: 清除缓存重试
